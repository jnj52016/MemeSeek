# MemeSeek 截图目录

这里保存 README 和求职演示使用的项目截图。

## 建议截图

| 文件 | 截图内容 | 截图时机 |
| --- | --- | --- |
| `meme-library.png` | 梗图列表页 | 展示搜索框、梗图卡片和上传入口 |
| `upload-drawer.png` | 上传抽屉 | 展示“点击、拖拽或 Ctrl+V 粘贴图片”以及图片预览 |
| `meme-detail.png` | 梗图详情弹窗 | 展示 AI 生成的标题、描述、标签和 OCR 结果 |
| `meme-edit.png` | 编辑梗图弹窗 | 展示标题、描述和标签编辑 |
| `meme-search.png` | 搜索结果 | 展示 URL 搜索参数和筛选结果 |
| `ai-settings.png` | AI 设置页 | 展示分析 AI、内容 AI 和本地配置说明 |

## 截图要求

- 优先使用浏览器全窗口截图，保留地址栏中的 `localhost:5173`，方便证明是可运行项目。
- 截图前隐藏真实 API Key，避免把密钥提交到仓库或发送给他人。
- 图片建议使用 PNG，宽度约 1440～1920 像素。
- 截图文件名使用上表名称，README 中的说明可以直接与文件对应。

## 推荐顺序

求职展示时按以下顺序展示：

1. `meme-library.png`：先说明产品目标。
2. `upload-drawer.png`：演示核心上传交互和 Ctrl+V 粘贴。
3. `meme-detail.png`：展示真实视觉模型分析结果。
4. `meme-edit.png`：展示信息编辑流程。
5. `meme-search.png`：展示搜索和 URL 参数同步。
6. `ai-settings.png`：说明配置隔离和本地安全边界。
