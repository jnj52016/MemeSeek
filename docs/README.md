# MemeSeek 项目文档

本目录统一保存 MemeSeek 的项目文档、设计稿和开发记录。

## 目录说明

```text
docs/
├─ frontend/       # 前端文档
├─ backend/        # 后端文档
├─ design/         # 页面原型、交互草图和视觉设计
├─ api/            # API、Swagger 和接口说明
├─ decisions/      # 重要技术决策和方案记录
└─ README.md       # 文档目录说明
```

## 文档命名建议

- 使用小写英文和短横线，例如 `project-structure.md`。
- 页面原型使用 `.html`，例如 `memeseek-mvp.html`。
- 前端相关文档放在 `frontend/`。
- 后端相关文档放在 `backend/`。
- 前后端共同使用的接口说明放在 `api/`。
